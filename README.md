Pour utiliser la calculatrice :
Entrez simplement les opérations que vous souhaitez effectuer.

Par exemple : <br>
15 *  2 + 4.5 *  9               

Donnera : <br>
15 * 2 + 4,5 * 9 <br>
 = 70,5

Vous pouvez ensuite continuer vos calculs à partir du résultat.
Les opérateurs existants sont : '+' | '-' | '*' | '/' | '^'
